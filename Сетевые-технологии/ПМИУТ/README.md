# ПМИУТ

Лабораторные работы по дисциплине «Промышленная маршрутизация и управление трафиком».

## Работы

- ЛР1 — Развертывание тестовой сети связи с использованием ContainerLab
- ЛР2 — Статическая маршрутизация между филиалами
- ЛР3 — Настройка OSPF, MPLS и EoMPLS
- ЛР4 — Настройка iBGP, L3VPN (VRF) и VPLS

## Используемые технологии

- MikroTik RouterOS
- ContainerLab
- Docker
- VLAN
- DHCP
- OSPF
- MPLS
- BGP
- VRF
- VPLS
