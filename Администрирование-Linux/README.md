# Администрирование Linux

Практические работы по администрированию Linux-систем и серверной инфраструктуры.

Основные темы:
- Linux
- Nginx
- HTTPS (TLS)
- PostgreSQL
- Docker и Docker Compose
- Резервное копирование
- Мониторинг и логирование
- Облачные сервисы
  
## Работы

- Linux 1 — основы Linux
- Linux 2 — работа с пользователями и правами доступа
- Linux 3 — сетевые настройки и сервисы
- Linux 4 — серверные службы
- Linux 5 — Docker и контейнеризация
- Linux 6 — Docker Compose и развёртывание приложений
