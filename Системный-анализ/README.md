## Краткое описание

В разделе представлены учебные проекты по системному анализу и проектированию информационных систем. Работы включают анализ предметной области, построение UML-диаграмм, DFD и IDEF-моделей, описание архитектуры системы, а также разработку функциональных и нефункциональных требований.
