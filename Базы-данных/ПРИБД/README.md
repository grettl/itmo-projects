# ПРИБД

Лабораторные работы по дисциплине «Проектирование и реализация баз данных».

## Работы

- ЛР1 — Установка PostgreSQL, создание базы данных HR и проектирование структуры БД
- ЛР2 — Основы SQL: SELECT, WHERE, ORDER BY, функции и условия
- ЛР3 — JOIN, агрегатные функции, GROUP BY и HAVING
- ЛР4 — Представления, материализованные представления, функции и процедуры
- ЛР5 — Транзакции, блокировки и уровни изоляции

## Используемые технологии

- PostgreSQL
- pgAdmin
- SQL
- ER-моделирование
- Представления (VIEW)
- Транзакции и блокировки
